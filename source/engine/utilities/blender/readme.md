## Installation

1. Install addon using one of the methods:
    Automatic method:
      * Rename the downloaded zipfile to `ray_scene`.
      * From Blender menu, navigate to `Edit -> Preferences`. Then `Add-ons` tab.
      * Click on the `Install` button and point it to downloaded zipfile.
    Manual method:
      * Extract zipfile to `ray_scene` directory which should be placed in following location:
      * Windows:\
        `%APPDATA%\Blender Foundation\Blender\2.*\scripts\addons`
      * Linux: \
        `~/.config/blender/2.*/scripts/addons`
      * *(If addon won't appear as a choice in the Blender's preferences, click `Refresh` button or restart the application.)*
2. Activate the addon in Blender preferences: toggle on the checkbox `[✓]` for `ray_scene` entry in Add-ons tab.
